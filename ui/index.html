Deployment blueprint

Yhteenveto kayttoliittyman vaiheista WordPressista julkaistuun sivustoon.

    Automation checklist

    • 1. Migraatio ajetaan CLI-backend kirjoittaa sisallot, raportin ja halutessa redirectit.
    • 2. Eleventy build Build-komento ajetaan GitHub Actionsissa.
    • 3. GitHub Pages deploy Push oletusbranchiin tuottaa automaattisen julkaisun.
    • 4. Optional domain CNAME ja DNS asetetaan vain jos oma domain halutaan mukaan.

    Tuloste

    API-vastaukset ja migraation raportti nakyvat tassa.
